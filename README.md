# My nvim configs

These are my nvim configs. They sometimes work on my system, they might also work on yours. To increase your changes, use NVIM v0.12.

The configs are very much inspired by this minimal Neovim config: [https://vieitesss.github.io/posts/Neovim-new-config/](https://vieitesss.github.io/posts/Neovim-new-config/)

My "live" configs are sometimes a mess so they'll remain private for now. I might occasionally update this repo to match my dev environment in case of any major changes.
